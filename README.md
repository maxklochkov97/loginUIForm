# Форма регистрации

Страница входа (Скриншот 1), тут можно ввести логин и пароль и войти в приложение (Скриншот 4). Если данных для входа не будет, то будет ошибка страница (Скриншот 2).
 
Внизу есть кнопка “Sing up” она перекинет на окно регистрации (Скриншот 3), там можно зарегистрировать новый аккаунт и сразу войти в приложение, либо нажать на кнопку “Sing in” и вернуться на предыдущую страницу. 
 
Если перезагрузить приложение когда пользователь авторизован, а потом запустить заново, то вход будет с рабочей страницы приложения (Скриншот 4).
 
Данные пользователя отправляются на firebase.google.com и там же хранятся.
